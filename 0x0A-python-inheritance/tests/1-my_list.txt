Using ``1-my_list``

------------------------------------------

MyList class inherits from list. It has a method print_sorted,
that sorts the list in an ascending sort.

------------------------------------------

Usage:
======

Import MyList from 1-my_list module
>>> MyList = __import__("1-my_list").MyList

For positive numbers:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4]


For negative numbers:
    >>> my_list.append(-2)
    >>> my_list.append(-5)
    >>> my_list.append(-1)
    >>> my_list.append(-9)
    >>> my_list.print_sorted()
    [-9, -5, -2, -1, 1, 2, 3, 4]

Test for decimals:
    >>> my_list.append(-92)
    >>> my_list.append(35.7)
    [-92, -9, -5, -2, -1, 1, 2, 3, 4, 35.7]
